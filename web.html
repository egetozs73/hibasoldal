Weboldalak készítése

Weboldalakat megtekinteni böngészőprogramokkal lehet, pl.:

  • Firefox ill. Firefox Developer Edition
  • Chrome
  • Opera
  • Microsoft Edge
  • IE (Internet Explorer)
  • Maxthon

Néhány böngészőprogram ikonja.

Weboldalakat létrehozni lehet:

  • egyszerű szövegszerkesztővel (nem dokumentumszerkesztő, mint a word) pl.: jegyzettömb, Notepad++
  • WYSIWYG weblapkészítő alkalmazásokkal (what you see is what you get) pl.: Sharepoint Designer, Dreamweaver, phpDesigner, UltraEdit, stb.
  • tartalomkezelő rendszerek használatával, pl.: WordPress, Drupal, Joomla, stb.

A weboldal szerkezete:

  1. a dokumentum típusának meghatározása
  2. fej rész, az adott HTML-oldalra vonatkozó meta-adatokat tartalmazza (adat az adatról...)
  3. törzs rész, maga az oldal tartalma
